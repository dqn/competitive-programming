# competitive-programming

## Run

```bash
$ g++ -Wall -std=c++14 <path>
$ ./a.out
# rm a.out
```

# Test

```bash
# Need `*.cpp` and `*.txt` in same directory
$ node test.js <path>
```

# Initialize for AtCoder

```bash
# Need `./template.cpp`
$ node init.js <url>
# => ./atcoder/abcXXX/x.cpp, ./atcoder/abcXXX/x.txt
```
