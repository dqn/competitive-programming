# competitive-programming

## Run

```bash
$ g++ -Wall -std=c++14 <path>
$ ./a.out
# rm a.out
```

## AtCoder

### Installation

See [atcoder-cli](https://github.com/dqn/atcoder-cli)

### Init

```bash
$ atcoder init <url>
```

### Test

```bash
$ atcoder test <contest> <task>
```

### Initialize for AtCoder

```bash
$ atcoder submit <contest> <task> -t # test before submit
```
